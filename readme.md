# event calender

## project setup

### to start the frontend
```sh
cd typescripttraining
```
```sh
npm install
```
```sh
npm run dev
```
### to start the backend
```sh
cd typescripttraining
```
```sh
npm run db
```
#### execution error?
run this command in a powershell as admin
```sh
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine
```
